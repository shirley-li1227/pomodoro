# Pomodoro Timer

一个基于 React + TypeScript + Vite 的番茄钟应用，采用淡雅橙红（番茄）主题。

## 已实现功能

- 默认工作 `25` 分钟，休息 `5` 分钟
- 支持 `开始`、`暂停`、`重置`
- 倒计时显示格式为 `MM:SS`
- 页面加载时请求浏览器通知权限
- 倒计时结束发送通知（标题：`番茄钟`，内容：`时间到了，该休息了！`）
- 倒计时结束播放提示音（音频文件位于 `public/alert.wav`）
- 记录完成番茄钟数量并显示：
  - 今日完成数量
  - 本周完成数量（按周一到周日）
- 数据通过 `LocalStorage` 持久化保存：
  - 任务标题：`pomodoro-task-title`
  - 任务记录：`pomodoro-task-records`
- 支持编辑任务标题，并在工作番茄结束时保存任务完成记录（含任务标题与完成时间）
- 页面展示最近任务记录列表（最新在前）
- 标题包含番茄图标，整体 UI 为淡雅橙红配色

## 启动方式

```bash
npm install
npm run dev
```
